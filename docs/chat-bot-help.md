# Moblin chat bot help

```
!moblin obs fix
Fix OBS input.

!moblin alert <name>
Trigger alerts. Configure alert names in alert widgets.

!moblin fax <url>
Fax the streamer images.

!moblin snapshot <optional message>
Take snapshot.

!moblin filter <filter> <on/off>
Turn a filter on or off.

  <filter> is movie, grayscale, sepia, triple, pixellate or 4:3.

  <on/off> is on or off.

!moblin map zoom out
Zoom out map widget temporarily.

!moblin tts on
Turn on chat text to speech.

!moblin tts off
Turn off chat text to speech.

!moblin say <message...>
Say given message.

!moblin tesla trunk open
Open the trunk.

!moblin tesla trunk close
Close the trunk.

!moblin tesla media next
Next track.

!moblin tesla media previous
Previous track.

!moblin tesla media toggle-playback
Toggle playback.
```
